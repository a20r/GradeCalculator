# GradeCalculator
Calculating your remaining grades so you don't have to 

# Running
    python calculator.py --grades=[your grades.json file] --goal=[your goal grade overall]

# Example
    $ python calculator.py --grades=grades.json --goal=16.5
    CP : 12.3555555556
    DM : 11.2222222222
    DS : 12.7555555556
